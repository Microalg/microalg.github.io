MicroAlg - Position de la puce

Définir une nouvelle commande

Dans l’échantillon de code ci-dessous, on a préparé du code MicroAlg qui :

  1. définit une nouvelle commande : Position_finale,
  2. affiche le résultat de cette nouvelle commande.

Testez ce code en remplaçant par exemple 0 par une autre valeur.

Travail à faire : En modifiant le code de cette nouvelle commande, faites en sorte qu’elle simule les 10 sauts de notre puce bien aimée.

Utilisation de la nouvelle commande

Recopier la commande que vous venez de créer dans l’échantillon de code ci-dessous, et utilisez-là pour écrire le programme demandé :

  • simulation d’un « grand nombre » de promenades aléatoires de 10 sauts ;
  • affichage de la fréquence des promenades pour lesquelles la position finale est 0.

Le « grand nombre » doit être au moins égal à 1000, mais ne doit pas être trop grand sous peine de « planter » MicroAlg.

Travail à faire : Utiliser la nouvelle commande pour compter le nombre de retours à l’origine sur un « grand nombre » de promenades de 10 sauts.

Nouvelle commande pré-définie

Même consigne, sauf que dans l’échantillon de code ci-dessous, il n’y a pas besoin de définir la commande Position_finale car elle a été définie pour vous.
