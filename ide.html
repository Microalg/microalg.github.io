MicroAlg - web IDE

Qu’est-ce ?

Cette page est un environnement de développement intégré de fortune pour MicroAlg.

Elle permet pour l’instant de faire quelques tests mais aussi d’avoir une vitrine en ligne pour ce langage.

D’autres façons d’utiliser MicroAlg sont en cours de développement. Il est prévu par exemple d’utiliser :

  • jsrepl pour obtenir un meilleur résultat dans une page web,
  • voire même, hors navigateur, un interpréteur plus efficace que celui de cette page, écrit en Javascript.

Instructions

Édition de scripts

La première zone de texte permet d’éditer des scripts. Pour les exécuter, appuyer sur Ctrl+Entrée.

Aucun résultat ne sera affiché sans utiliser la commande Afficher. L’affichage se fait juste après la fenêtre d’édition.

Console interactive

La deuxième zone de texte permet un échange « instructions après instructions » avec l’environnement.

C’est pratique pour tester des instructions quand on développe un programme dans la zone d’édition de scripts.
