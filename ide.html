MicroAlg - web IDE

Qu’est-ce ?

Cette page est un environnement de développement intégré de fortune pour MicroAlg.

Elle permet pour l’instant de faire quelques tests, mais aussi d’avoir une vitrine en ligne pour ce langage. D’autres façons d’utiliser MicroAlg sont en cours de développement. Voir pour cela la page d’installation.

Instructions

Édition de scripts

La zone avec les pièces de puzzle permet de programmer visuellement, avec la souris. Fini les fautes de frappe. Elle crée le code MicroAlg correspondant et l’envoie dans la zone de texte juste en dessous.

Commencer par cliquer sur une des catégories de commandes sur la gauche (Valeurs, Cmdes……) et glisser un bloc Afficher dans le bloc Programme de l’espace de travail. Puis dans la catégorie Autres, prendre un bloc de texte (avec les guillemets "..."), le glisser au bout du bloc Afficher puis taper un message dans ce dernier bloc. Le code correspondant s’affichera dans la fenêtre en dessous, et pourra être exécuté en cliquant sur le bouton OK.

Pour apprendre MicroAlg au travers de cette interface, suivre le tutoriel correspondant.

La première zone de texte permet d’éditer des scripts. Pour les exécuter, appuyer sur Ctrl+Entrée ou cliquer sur le bouton OK.

L’affichage des résultats se fait juste après la fenêtre d’édition si vous utilisez la commande Afficher, ou dans la fenêtre graphique si vous utilisez des commandes graphiques.

Consoles interactives

Les autres zones de texte permettent un échange « instructions après instructions » avec l’environnement.

C’est pratique pour tester des instructions quand on développe un programme dans la zone d’édition de scripts.

En particulier, la deuxième zone de texte est un terminal de fortune, bricolé pour l’occasion. Pour exécuter le code, appuyer sur Ctrl+Entrée ou cliquer sur le bouton OK.

La troisième zone de texte utilise un plugin jQuery d’émulation de terminal. Elle est donc assez confortable mais ses parenthèses ne sont pas colorées et on ne peut pas passer à la ligne sans exécuter le code.
